# GitHub Upload Checklist

Before making the repository public, confirm the following:

- [ ] No `.dta`, `.sav`, `.xlsx`, `.zip`, or raw SIHBS microdata files are inside `data/raw/`.
- [ ] No household-level predicted-risk file is inside `outputs/`.
- [ ] No private file with names such as `analysis_household_private`, `predicted_risk_private`, or `REAL` is included.
- [ ] Only aggregated outputs, figures, code, documentation, metadata, and schema-only files are public.
- [ ] Update `CITATION.cff` and the README with your real GitHub repository URL.
- [ ] Add repository topics: `bayesian-statistics`, `climate-risk`, `food-security`, `somalia`, `early-warning`, `humanitarian-data`, `targeting-simulation`.

Recommended public repository name:

```text
who-would-we-miss-somalia
```
