# Data Ethics and Public-Sharing Note

This repository is designed to be public-safe. The official SIHBS 2022 microdata are not redistributed here.

The repository includes only:

- reproducible scripts,
- aggregated summary outputs,
- public figures,
- methodology documentation,
- metadata inventory,
- schema-only processed data.

Household-level predictions and raw microdata should remain private. The workflow is intended as an analytical demonstration of Bayesian risk targeting, not an operational beneficiary-selection tool.
